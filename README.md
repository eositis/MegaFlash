# MegaFlash for Apple IIc/IIc+

## Apple Firmware

The firmware is developed with `CC65`.

There is not enough space in Apple ROM to store the Control Panel program. So, the Control Panel is a separate program written in C. The compiled binary is stored on Pico. 

`firmware` Project directory of Apple Firmware Patches\
`cpanel` Control Panel project\
`common` Files used by both firmware and cpanel projects
